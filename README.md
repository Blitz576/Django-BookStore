## How To Start
### Requirements
- virtualenv library
- to install it run on your system
```
pip install virtualenv
```

- create your environment on the directory  
```
cd Django-BookStore
python -m venv venv

#Activate your virtual environment
## Windows

venv\Scripts\activate

## linux\mac

source ./venv/Scripts

# install Requirements

pip install -r requirements.txt


```
